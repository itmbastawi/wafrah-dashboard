# Inventory Dashboard (scaffold)

This workspace contains a scaffold of the Next.js inventory dashboard described in `tasks.md`.

Quick start:

1. Install dependencies

```bash
cd /home/mohbastawi/projects/wafra
npm install
```

2. Run dev server

```bash
npm run dev
```

Notes:
- This is a TypeScript + Next.js (app directory) scaffold with Tailwind CSS.
- The app expects a backend API at `http://localhost:8000` by default. Set `NEXT_PUBLIC_API_URL` to change.
